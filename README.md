## **Script**

### **Server Start**

```bash
# watch mode
$ npm run start:dev

# production mode
$ npm run build
$ npm run start:prod
```

### **Test**

```bash
# unit test
$ npm run test

# watch mode
$ npm run test:watch

# e2e test
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
